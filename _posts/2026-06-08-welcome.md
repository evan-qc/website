This is a **placeholder post** created to test and validate the new Jekyll
layouts. The body is written in Markdown, and Jekyll converts it to HTML at
build time.

## What this validates

- The `post` layout renders the title, date, and body
- The `post` layout correctly nests inside `default` (shared nav + footer)
- New posts show up automatically on the [blog index](/journal.html)

## Things you can do in a post

You can write normal prose, add **bold** and _italic_ text, and link to
other pages like our [case studies](/case-studies.html).

1. Ordered lists work
2. So do unordered lists
3. And inline `code`

> Blockquotes render too — handy for pulling out a client quote.

To publish another post, drop a new Markdown file in `_posts/` named with the
required `YYYY-MM-DD-title.md` format. That's it.
